Программа на основе данных из xlsx-файла, создает набор готовых команд с измененными параметрами.

Файл command_template.txt хранит в себе шаблоны некоторых команд.
Файл regex_template.txt хранит список шаблонов, на основе которых ищутся совпадения между изменяемыми параметрами команд и исходными данными. 
В папке files находятся xlsx-файлы с исходными данными.
В папке gen_scripts создаются файлы с готовым набором команд.
